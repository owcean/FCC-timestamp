Timestamp Microservice

Enter a date string, a Unix timestamp (in milliseconds), or leave blank for the current time.

Try these examples:

  • Date String: 2015-12-25
  • Unix Timestamp: 1451001600000
  • Empty (Current Time)

API Response (JSON)

{
    "unix": "...",
    "utc": "..."
}

Functionality mimics the API endpoint logic for freeCodeCamp's Timestamp Microservice challenge. KGT.
